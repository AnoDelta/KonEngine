# Roadmap

KonEngine is still in its early stages. Here's a rough idea of where it's headed.

## Current Focus
- 2D renderer improvements
  - Texture/sprite rendering
  - Text rendering
  - More primitives

## Planned
- Input system (keyboard/mouse)
- Audio
- Better window management

## Far Future
- GUI editor — to make working with the engine easier without touching code directly

## No Guarantees
This is a personal project built for my own use and for friends. 
Features get added when I need them. Nothing here is a promise.
